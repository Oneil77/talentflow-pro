# 🚀 TalentFlow Pro — AI-агент для подбора персонала

AI-агент, который анализирует рынок труда, парсит резюме и вакансии, сравнивает кандидатов и даёт рекомендации по найму.

## 🎯 Возможности

- **Анализ рынка** — реальные данные с hh.ru, топ-навыки, зарплаты, тренды
- **Парсинг резюме** — извлечение навыков, опыта, проектов из PDF/DOCX/TXT
- **Анализ вакансий** — извлечение требований из вакансий hh.ru
- **Сравнение кандидата с вакансией** — Match Score, рекомендации, gap-навыки
- **Интерактивный дашборд** — графики спроса, зарплат, трендов

## 🚀 Как запустить

```bash
# Клонировать репозиторий
git clone https://github.com/Oneil77/talentflow-pro.git
cd talentflow-pro

# Установить зависимости
pip install -r requirements.txt

# Запустить бэкенд
cd backend
uvicorn main:app --reload --host 0.0.0.0 --port 8000

# В отдельном терминале запустить фронтенд
cd frontend
python -m http.server 8080
