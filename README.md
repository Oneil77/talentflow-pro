# 🚀 TalentFlow Pro — AI-агент для подбора персонала

[![Python](https://img.shields.io/badge/Python-3.12-blue.svg)](https://www.python.org/)
[![FastAPI](https://img.shields.io/badge/FastAPI-0.104-green.svg)](https://fastapi.tiangolo.com/)
[![License](https://img.shields.io/badge/License-MIT-yellow.svg)](LICENSE)

> AI-агент, который анализирует рынок труда, парсит резюме и вакансии, сравнивает кандидатов и даёт рекомендации по найму.

## 🎯 Возможности

| Функция | Описание |
|---------|----------|
| **Анализ рынка** | Реальные данные с hh.ru, топ-навыки, зарплаты, тренды |
| **Парсинг резюме** | Извлечение навыков, опыта, проектов из PDF/DOCX/TXT |
| **Анализ вакансий** | Извлечение требований из вакансий hh.ru |
| **Сравнение кандидата с вакансией** | Match Score, рекомендации, gap-навыки, AI-инсайты |
| **Интерактивный дашборд** | Графики спроса, зарплат, трендов |

## 🚀 Быстрый старт
### 1. Клонирование репозитория
```bash
git clone https://github.com/Oneil77/talentflow-pro.git
cd talentflow-pro

2. Создание виртуального окружения и установка зависимостей
bash
python3 -m venv venv
source venv/bin/activate  # на Windows: venv\Scripts\activate
pip install -r requirements.txt

3. Запуск бэкенда
bash
cd backend
uvicorn main:app --reload --host 0.0.0.0 --port 8000
Сервер будет доступен по адресу: http://localhost:8000

4. Запуск фронтенда (в отдельном терминале)
bash
cd frontend
python3 -m http.server 8080
5. Открыть в браузере
Главная страница: http://localhost:8080
Дашборд: http://localhost:8080/dashboard.html

🧠 Как работает AI-агент
1. Анализ кандидата (ручной ввод)
Вводишь навыки, опыт, образование → агент выдаёт Match Score и рекомендации.
2. Анализ резюме (загрузка файла)
Загружаешь PDF/DOCX/TXT → агент сам извлекает навыки, опыт, проекты.
3. Анализ под вакансию
Вставляешь ссылку на вакансию hh.ru + загружаешь резюме → агент сравнивает и выдаёт результат.

📊 Дашборд
Топ-10 навыков — самые востребованные компетенции
Зарплаты по ролям — средние зарплаты AI-специалистов
Распределение зарплат — как распределены зарплаты на рынке
Рост ролей — динамика спроса на разные профессии

🛠️ Технологии
Backend: Python, FastAPI, Uvicorn, Requests, BeautifulSoup, PyPDF2, python-docx
Frontend: HTML5, CSS3, JavaScript, Chart.js
API: hh.ru API
NLP: pymorphy2 (для улучшенного извлечения навыков)
